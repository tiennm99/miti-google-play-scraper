# google-play-scraper-api

> **Deprecated.** Prefer the unified [store-scraper-api](https://github.com/tiennm99/store-scraper-api), which hosts both Google Play and Apple App Store scrapers in a single deployment. This repo stays available for users who only want the Google Play endpoints in their own deployment.

Host [google-play-scraper](https://github.com/facundoolano/google-play-scraper) on [Vercel](https://vercel.com).

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/tiennm99/google-play-scraper-api)

## Usage

POST requests to `/:method` with raw JSON body parameters.

See [google-play-scraper documentation](https://github.com/facundoolano/google-play-scraper) for method parameters.
